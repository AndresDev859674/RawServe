# RawServe
Serve any raw HTML file directly in your browser without hosting.

# So why?
There are people like me who don't want to host, manage a domain, or anything else, but rather serve a pre-built website. You can simply place a URL containing HTML and other elements anywhere (it could be localhost or an IP address) and you're done! Or you can simply create a repository and link the HTML to it. For example:

https://raw.githubusercontent.com/Hello/My-Web/refs/heads/main/index.html
