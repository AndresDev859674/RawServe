Welcome to RawServe

RawServe is a lightweight, zero-hosting web importer designed to fetch and render raw HTML pages directly inside your browser. By resolving dependency paths natively on-the-fly, you can instantly test and display your files without deployment pipelines or CORS headaches.

Our Main Features is :

  • It's Fast
  • Simple UI, No Bullshit, Just Works. (And retro)
  • 100% Free and Open Source
  • Absolute Privacy, They are actually URLs that provide a view of RAW html and nothing else

And Can do :

  • Directly serve raw HTML files from GitHub, GitLab, or any public URL.
  • Automatically resolve relative paths for assets like CSS, JS, and images.
  • Support for local HTML file uploads for instant rendering.
  • Choose between different CDN providers for optimized delivery.
  • Advanced diagnostics console for troubleshooting and debugging.
